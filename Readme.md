# Exam Preparation Platform

A modern, single-page web application featuring a Glassmorphism UI that allows BCS students to take subject-wise quizzes, validate credentials securely, and track their performance using local storage.

# Starting the WebApp

1. Terminal Launch: Open Terminal in that folder and type start index.html.
2. Manual Launch: Double-click index.html to open in any browser.
3. Login: Use a valid email and strong password (8+ chars, 1 Cap, 1 Num).
4. Start: Click a subject card to begin and view history below.
